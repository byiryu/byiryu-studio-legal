# byiryu-studio-legal

byiryu studio 가 운영하는 모바일 앱들의 약관·개인정보처리방침 공개 호스팅.

## 공개 URL (GitHub Pages)

### PALETI

- [개인정보 처리방침](https://byiryu.github.io/byiryu-studio-legal/paleti-privacy.html)
- [이용약관](https://byiryu.github.io/byiryu-studio-legal/paleti-terms.html)

## 정책

각 앱별 약관·개인정보처리방침은 **self-contained** 으로 작성됩니다 — 운영자 정보·사용자 권리·분쟁 해결 등 공통 항목을 각 문서 안에 직접 포함. 사용자가 단일 페이지에서 전체 내용 확인 가능.

## 갱신 워크플로우

원본 Markdown 은 [ApplicationProject `_docs/legal/`](https://github.com/byiryu/ApplicationProject) (Private) 에 보관. 변경 시:

1. `_docs/legal/{app}-*.md` 편집
2. Python markdown 으로 HTML 변환
3. 본 repo 에 push
4. GitHub Pages 자동 빌드 (~30초)

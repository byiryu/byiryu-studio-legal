PALETI 회원 탈퇴 안내

서비스명: PALETI (한국 모바일 앱 — 색으로 시작하는 하루)
운영자: byiryu studio
문의 이메일: byiryustudio@gmail.com
시행일: 2026-05-19

본 안내는 Google Play 데이터 보안 정책 및 Apple App Store Connect 의 "사용자 계정 삭제 (Account Deletion)" 요구사항에 따라 제공됩니다. PALETI 는 사용자가 본인의 계정과 모든 관련 데이터의 삭제를 요청할 수 있는 두 가지 방법을 제공합니다.

1. 회원 탈퇴 방법

방법 1 — 앱 내 탈퇴 (즉시 처리, 권장)

  1. PALETI 앱 실행 후 Google 또는 Apple 계정으로 로그인합니다.
  2. 메인 화면 우측 상단의 설정 아이콘 을 탭해 마이페이지로 진입합니다.
  3. 화면 하단 "계정" 섹션의 [회원 탈퇴] 항목을 탭합니다.
  4. 확인 dialog 의 [탈퇴] 버튼을 탭합니다.
  5. 계정과 모든 데이터가 즉시 삭제되며, 앱은 로그인 화면으로 돌아갑니다.

본 흐름은 서버의 회원 탈퇴 API 를 호출하여 계정과 관련 데이터를 데이터베이스에서 즉시 영구 삭제합니다.

방법 2 — 이메일 요청 (영업일 기준 7일 이내 처리)

앱 접근이 어려운 경우 (앱 삭제 후 / 디바이스 분실 등) 다음 양식으로 이메일을 보내주세요.

  • 받는 사람: byiryustudio@gmail.com
  • 제목: "PALETI 회원 탈퇴 요청"
  • 본문에 포함할 내용:
    • 가입 시 사용한 Google 또는 Apple 계정 이메일 주소
    • 본인 확인을 위한 추가 정보 (요청 시 안내)

이메일 수신 후 본인 확인 절차를 거쳐 영업일 기준 7일 이내에 계정과 데이터를 삭제합니다.

2. 삭제되는 데이터 (즉시 삭제, 복구 불가)

회원 탈퇴 시 다음 데이터가 즉시 영구 삭제되며 복구할 수 없습니다.

  • 계정 식별 정보 (이메일 주소, 사용자 UUID)
  • 사용자 환경설정
    • 선호 컬러 테마 (10종 중 선택)
    • 다크/라이트 모드 설정
    • 알림 토글 (아침 푸시·미세먼지 경보)
    • 기본 위치 (수동 선택 도시)
    • 성별 정보
  • 추천 이력 (사용자가 받은 모든 코디 추천 기록)
  • FCM 푸시 토큰 (푸시 알림 발송용 기기 식별자)

3. 익명 캐시로 일시 보관되는 데이터 (최대 7일)

회원 탈퇴 후에도 다음 데이터는 익명 캐시 형태로 일시 보관될 수 있습니다. 이 데이터는 사용자 본인을 식별하지 않습니다.

  • 위치 격자 좌표 기반 날씨·미세먼지 캐시 (5km × 5km 격자 단위)
  • 추천 결과 캐시 (사용자 식별 정보 제거됨)

본 데이터는 외부 공공 API (기상청 단기예보·한국환경공단 에어코리아) 의 호출 효율을 위한 익명 캐시입니다. 사용자 UUID 또는 이메일과의 연결이 끊긴 상태로 격자 좌표 단위로만 저장되며, 최대 7일 후 자연 만료됩니다.

4. 변경 고지

본 안내가 변경될 경우 본 페이지에서 시행일과 함께 게시됩니다. 중요한 변경은 앱 내 공지 또는 푸시 알림으로 별도 안내됩니다.
